nimble.calculate.inverse¶

nimble.calculate.inverse(aObj)¶

Compute the (multiplicative) inverse of a nimble Base object.

Parameters

aObj (nimble Base object.) – Square object to be inverted.

Returns

aInv (nimble Base object.) – Inverse of the object aObj

Raises

  • InvalidArgumentType – If aObj is not a nimble Base Object. If aObj elements types are not supported.

  • InvalidArgumentValue – If aObj is not square. If aObj is not invertible (Singular).

Examples

>>> lst = [[1, 2], [3, 4]]
>>> X = nimble.data(lst)
>>> X
<Matrix 2pt x 2ft
     0 1
   ┌────
 0 │ 1 2
 1 │ 3 4
>
>>> inverse(X)
<Matrix 2pt x 2ft
       0      1
   ┌──────────────
 0 │ -2.000 1.000
 1 │ 1.500  -0.500
>
