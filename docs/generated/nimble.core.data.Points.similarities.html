Points.similarities¶

Points.similarities(similarityFunction)¶

Calculate similarities between points.

Return a new object containing the results of the similarityFunction.

Parameters

similarityFunction (str) – The name of the function. The accepted strings include: ‘correlation’, ‘covariance’, ‘sample covariance’, ‘population covariance’ and, ‘dot product’. Pearson correlation coefficients are used for ‘correlation’.

Returns

nimble Base object

See also

statistics, nimble.calculate.similarity

Keywords: correlation, covariance, sample covariance, population covariance, dot product, similarity, relationship, cor, pearson, spearman, correlation coefficient
