nimble.match.floating¶

nimble.match.floating(value)¶

Determine if a value is a float type.

Return True if the value is a float type, otherwise return False.

Parameters

value –

Returns

bool

See also

anyFloating, allFloating

Examples

>>> floating(3.0)
True
>>> floating(3)
False
>>> floating(True)
False
