nimble.calculate.residuals¶

nimble.calculate.residuals(toPredict, controlVars)¶

Calculate the residuals by a linear regression model.

Parameters

  • toPredict (nimble Base object) – Each feature will be used as the independent variable in a separate linear regression model with the controlVars as the dependent variables.

  • controlVars (nimble Base object) – Must have the same number of points as toPredict. Each point will be used as the dependant variables to do predictions for the corresponding point in toPredict.
