nimble.match.allNonMissing¶

nimble.match.allNonMissing(data)¶

Determine if all values in the data are missing.

Return True if every value in the data is considered to be missing. Missing values in Nimble are None and (python or numpy) nan values.

Parameters

data (Nimble Base object) –

Returns

bool

See also

anyNonMissing, nonMissing

Examples

>>> lst = [[1, 1, 3.0],
...        [2, 0, 2.0],
...        [3, -1, 1.0]]
>>> X = nimble.data(lst)
>>> allNonMissing(X)
True

>>> lst = [[1, 1, 3.0],
...        [2, float('nan'), 2.0],
...        [3, -1, 1.0]]
>>> X = nimble.data(lst)
>>> allNonMissing(X)
False
